  • Dnaseq standard

WARNING: this workflow is experimental! Use at your own risk!

dnaseq_standard_experimental

description
Aligns DNA reads using bwa mem
outputs
{'harmonized_bam': 'Harmonized DNA-Seq BAM, aligned with bwa mem'}
allowNestedInputs
true

Inputs

Required

  • bam (File, required): Input BAM to realign
  • bwa_db (File, required): Gzipped tar archive of the bwa reference files. Files should be at the root of the archive.
  • bwa_mem._runtime (Any, required)
  • get_read_groups._runtime (Any, required)
  • sort._runtime (Any, required)
  • validate_input_bam._runtime (Any, required)
  • bam_to_fastqs.bam_to_fastq._runtime (Any, required)
  • bam_to_fastqs.fqlint._runtime (Any, required)
  • bam_to_fastqs.quickcheck._runtime (Any, required)
  • bam_to_fastqs.split._runtime (Any, required)
  • samtools_merge.basic_merge._runtime (Any, required)
  • samtools_merge.final_merge._runtime (Any, required)
  • samtools_merge.inner_merge._runtime (Any, required)

Optional

  • validate_input_bam.reference_fasta (File?)
  • samtools_merge.basic_merge.new_header (File?)
  • samtools_merge.final_merge.new_header (File?)
  • samtools_merge.inner_merge.new_header (File?)

Defaults

  • prefix (String, default=basename(bam,".bam")): Prefix for the BAM file. The extension .bam will be added.
  • use_all_cores (Boolean, default=false): Use all cores? Recommended for cloud environments.
  • validate_input (Boolean, default=true): Ensure input BAM is well-formed before beginning harmonization?
  • bwa_mem.modify_disk_size_gb (Int, default=0)
  • bwa_mem.ncpu (Int, default=2)
  • bwa_mem.prefix (String, default=sub(basename(read_one_fastq_gz),"([_\.][rR][12])?(\.subsampled)?\.(fastq|fq)(\.gz)?$","")): Prefix for the BAM file. The extension .bam will be added.
  • get_read_groups.modify_disk_size_gb (Int, default=0)
  • samtools_merge.max_length (Int, default=100)
  • sort.memory_gb (Int, default=25)
  • sort.modify_disk_size_gb (Int, default=0)
  • sort.prefix (String, default=basename(bam,".bam") + ".sorted"): Prefix for the BAM file. The extension .bam will be added.
  • sort.sort_order (String, default="coordinate")
  • validate_input_bam.ignore_list (Array[String], default=[])
  • validate_input_bam.index_validation_stringency_less_exhaustive (Boolean, default=false)
  • validate_input_bam.max_errors (Int, default=2147483647)
  • validate_input_bam.memory_gb (Int, default=16)
  • validate_input_bam.modify_disk_size_gb (Int, default=0)
  • validate_input_bam.outfile_name (String, default=basename(bam,".bam") + ".ValidateSamFile.txt")
  • validate_input_bam.succeed_on_errors (Boolean, default=false)
  • validate_input_bam.succeed_on_warnings (Boolean, default=true)
  • validate_input_bam.summary_mode (Boolean, default=false)
  • bam_to_fastqs.bam_to_fastq.F (String, default="0x900")
  • bam_to_fastqs.bam_to_fastq.G (String, default="0")
  • bam_to_fastqs.bam_to_fastq.append_read_number (Boolean, default=true)
  • bam_to_fastqs.bam_to_fastq.f (String, default="0")
  • bam_to_fastqs.bam_to_fastq.fail_on_unexpected_reads (Boolean, default=false)
  • bam_to_fastqs.bam_to_fastq.fast_mode (Boolean, default=true)
  • bam_to_fastqs.bam_to_fastq.modify_disk_size_gb (Int, default=0)
  • bam_to_fastqs.bam_to_fastq.modify_memory_gb (Int, default=0)
  • bam_to_fastqs.bam_to_fastq.ncpu (Int, default=2)
  • bam_to_fastqs.bam_to_fastq.output_singletons (Boolean, default=false)
  • bam_to_fastqs.bam_to_fastq.prefix (String, default=basename(bam,".bam")): Prefix for the BAM file. The extension .bam will be added.
  • bam_to_fastqs.bam_to_fastq.store_collated_bam (Boolean, default=false)
  • bam_to_fastqs.fqlint.disable_validator_codes (Array[String], default=[])
  • bam_to_fastqs.fqlint.modify_disk_size_gb (Int, default=0)
  • bam_to_fastqs.fqlint.modify_memory_gb (Int, default=0)
  • bam_to_fastqs.fqlint.paired_read_validation_level (String, default="high")
  • bam_to_fastqs.fqlint.panic (Boolean, default=true)
  • bam_to_fastqs.fqlint.single_read_validation_level (String, default="high")
  • bam_to_fastqs.quickcheck.modify_disk_size_gb (Int, default=0)
  • bam_to_fastqs.split.modify_disk_size_gb (Int, default=0)
  • bam_to_fastqs.split.ncpu (Int, default=2)
  • bam_to_fastqs.split.prefix (String, default=basename(bam,".bam")): Prefix for the BAM file. The extension .bam will be added.
  • bam_to_fastqs.split.reject_unaccounted (Boolean, default=true)
  • samtools_merge.basic_merge.attach_rg (Boolean, default=true)
  • samtools_merge.basic_merge.combine_rg (Boolean, default=true)
  • samtools_merge.basic_merge.modify_disk_size_gb (Int, default=0)
  • samtools_merge.basic_merge.name_sorted (Boolean, default=false)
  • samtools_merge.basic_merge.ncpu (Int, default=2)
  • samtools_merge.basic_merge.region (String, default="")
  • samtools_merge.final_merge.modify_disk_size_gb (Int, default=0)
  • samtools_merge.final_merge.name_sorted (Boolean, default=false)
  • samtools_merge.final_merge.ncpu (Int, default=2)
  • samtools_merge.final_merge.region (String, default="")
  • samtools_merge.inner_merge.attach_rg (Boolean, default=true)
  • samtools_merge.inner_merge.combine_rg (Boolean, default=true)
  • samtools_merge.inner_merge.modify_disk_size_gb (Int, default=0)
  • samtools_merge.inner_merge.name_sorted (Boolean, default=false)
  • samtools_merge.inner_merge.ncpu (Int, default=2)
  • samtools_merge.inner_merge.region (String, default="")

Outputs

  • harmonized_bam (File)
