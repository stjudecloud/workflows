scRNA-Seq Standard

This WDL workflow runs the Cell Ranger scRNA-Seq alignment workflow for St. Jude Cloud.

The workflow takes an input BAM file and splits it into FASTQ files for each read in the pair. The read pairs are then passed through Cell Ranger to generate a BAM file and perform quantification. Strandedness is inferred using ngsderive. File validation is performed at several steps, including immediately preceeding output.

Inputs

Required

  • bam (File, required): Input BAM format file to quality check
  • gtf (File, required): Gzipped GTF feature file
  • transcriptome_tar_gz (File, required): Database of reference files for Cell Ranger. Can be downloaded from 10x Genomics.
  • compute_checksum._runtime (Any, required)
  • count._runtime (Any, required)
  • strandedness._runtime (Any, required)
  • subsample._runtime (Any, required)
  • validate_bam._runtime (Any, required)
  • validate_input_bam._runtime (Any, required)
  • cell_ranger_bam_to_fastqs.bamtofastq._runtime (Any, required)
  • cell_ranger_bam_to_fastqs.fqlint._runtime (Any, required)
  • cell_ranger_bam_to_fastqs.quickcheck._runtime (Any, required)

Optional

  • validate_bam.reference_fasta (File?)
  • validate_input_bam.reference_fasta (File?)

Defaults

  • subsample_n_reads (Int, default=-1): Only process a random sampling of n reads. <=0 for processing entire input BAM.
  • use_all_cores (Boolean, default=false): Use all cores for multi-core steps?
  • validate_input (Boolean, default=true): Ensure input BAM is well-formed before beginning harmonization?

  • cell_ranger_bam_to_fastqs.gemcode (Boolean, default=false)
