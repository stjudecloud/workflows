gatk_reference

description
Fetches reference files for GATK.
outputs
{'fasta': 'FASTA file for the reference genome.', 'fasta_index': 'Index for the FASTA file for the reference genome.', 'fasta_dict': 'Sequence dictionary for the reference genome.', 'dbSNP_vcf': 'dbSNP VCF file for the reference genome.', 'dbSNP_vcf_index': 'Index for the dbSNP VCF file for the reference genome.', 'interval_list': 'List of intervals that will be used when computing variants.', 'known_vcfs': 'VCF files with known variants to use with variant calling.'}
allowNestedInputs
true

Inputs

Required

  • dbSNP_vcf_disk_size_gb (Int, required): Disk size (in GB) to allocate for downloading the dbSNP VCF file.
  • dbSNP_vcf_name (String, required): Name of the dbSNP VCF file.
  • dbSNP_vcf_url (String, required): URL from which to retrieve the dbSNP VCF file.
  • known_vcf_disk_size_gb (Int, required): Disk size (in GB) to allocate for downloading the VCF files with known variants.
  • known_vcf_names (Array[String], required): Names of the VCF files with known variants. Order should match that of known_vcf_urls.
  • known_vcf_urls (Array[String], required): URLs from which to retrieve VCF files with known variants.
  • reference_fa_disk_size_gb (Int, required): Disk size (in GB) to allocate for downloading the reference FASTA file.
  • reference_fa_md5 (String, required): MD5 checksum for the reference FASTA file.
  • reference_fa_name (String, required): Name of the output reference FASTA file.
  • reference_fa_url (String, required): URL from which to retrieve the reference FASTA file.
  • create_sequence_dictionary._runtime (Any, required)
  • dbsnp._runtime (Any, required)
  • dbsnp_index._runtime (Any, required)
  • faidx._runtime (Any, required)
  • fasta_download._runtime (Any, required)
  • intervals._runtime (Any, required)
  • known_vcf._runtime (Any, required)

Optional

  • dbSNP_vcf_index_name (String?): Name of the index for the dbSNP VCF file.
  • dbSNP_vcf_index_url (String?): URL from which to retrieve the index for the dbSNP VCF file.
  • interval_list_name (String?): Name of the list of intervals to use when computing variants.
  • interval_list_url (String?): URL from which to retrieve the list of intervals to use when computing variants.
  • create_sequence_dictionary.assembly_name (String?)
  • create_sequence_dictionary.fasta_url (String?)
  • create_sequence_dictionary.species (String?)
  • dbsnp.md5sum (String?)
  • dbsnp_index.md5sum (String?)
  • intervals.md5sum (String?)
  • known_vcf.md5sum (String?)

Defaults

  • dbSNP_vcf_index_disk_size_gb (Int, default=1): Disk size (in GB) to allocate for downloading the index for the dbSNP VCF file.
  • interval_list_disk_size_gb (Int, default=1): Disk size (in GB) to allocate for downloading the list of intervals to use when computing variants.
  • create_sequence_dictionary.memory_gb (Int, default=16)
  • create_sequence_dictionary.modify_disk_size_gb (Int, default=0)
  • create_sequence_dictionary.outfile_name (String, default=basename(fasta,".fa") + ".dict")
  • faidx.modify_disk_size_gb (Int, default=0)

Outputs

  • fasta (File)
  • fasta_index (File)
  • fasta_dict (File)
  • dbSNP_vcf (File?)
  • dbSNP_vcf_index (File?)
  • interval_list (File?)
  • known_vcfs (Array[File])
