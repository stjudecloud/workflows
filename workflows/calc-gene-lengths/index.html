calc_gene_lengths

Inputs

Required

  • gtf (File, required): GTF feature file

Optional

  • max_retries (Int?): Number of times to retry failed steps. Overrides task level defaults.

Defaults

  • calc.memory_gb (Int, default=8)
  • calc.modify_disk_size_gb (Int, default=0)
  • calc.outfile_name (String, default=basename(gtf,".gtf.gz") + ".genelengths.txt")

Outputs

  • gene_lengths (File)
